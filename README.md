# Instance_AMFSP
The instance and input data that support the research of additive manufacturing fuzzy scheduling.
AMPP_instances:The first instance dataset that contains 30 instances across six scales.
NASI_instances:The second instance dataset that contains 40 instances across eight scales.
machine_info:Information of SLM printers and post-processing machines.
Prepared_model:Models after preparation, which contains figures, angles, binary matrices and pointcloud files.
